# 💧 Calculadora de Água

Aplicativo mobile desenvolvido em React Native para cálculo de conta de água com base em faixas de consumo.

## Funcionalidades

- Cálculo de água e esgoto
- Taxa de regulação
- Detalhamento da conta
- Análise de consumo
- Dicas de economia
- Gráfico comparativo

## Tecnologias

- React Native (Expo)
- JavaScript

## Objetivo

Projeto desenvolvido para atividade de extensão com foco em educação ambiental e consumo consciente de água.
